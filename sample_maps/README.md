# Sample Maps

This folder contains lightweight PT/NI map examples from the two datasets used in development:

- `PIMEGA_gridscan_17_54_21`
- `PIMEGA_gridscan_00_47_31`

For each dataset, included files are:

- `map_pt_roi_power.npy`
- `map_pt_roi_power.png`
- `map_ni_score.npy`
- `map_ni_score.png`

These are intended for quick testing of:

- `part1_map_viewer/view_map_with_1d.py`
- `part1_map_viewer/launch_map_viewer_gui.py`

They are not full raw datasets.
